Lab 3: Troubleshooting

Lab 3 - Troubleshooting

  1. Display running container type the command docker container ls

     docker container ls

  2. Display all containers whether running or not, type command docker ps -a

     docker ps -a

  3. Display container logs, type the command docker logs [container ID or name]

     docker logs [conainter ID or name]

  4. Display running process in Windows OS, type command tasklist
